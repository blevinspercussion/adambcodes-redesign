The Automated Motivator

Positive Vibes Daily is a Facebook page that is powered by a Python program that I call The Automated Motivator (TAM). TAM runs three times per day, creating a motivational message against a background image before uploading the resulting image to the Facebook page.

TAM uses the OpenAI API to generate the background image, as well as the motivational message, by feeding a pseudo-randomly generated text prompt into the API. The text overlay and the watermark are handled by the PIL library, while the Facebook functionality is handled by the Facebook SDK Python library.
